# react-isomorphic
React Isomorphic

确保已安装nodejs，然后全局安装yarn用以代替npm
> npm install -g yarn

安装依赖包
> yarn install

全局安装nodemon、eslint、cross-env
> yarn global add nodemon eslint

打包client文件(PWA mode: devcp)
> yarn run devc

打包server文件(PWA mode: devsp)
> yarn run devs

启动server
> yarn run serve
